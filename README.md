## Setup
This project is running on Node js environment, so you have to setup it before use the project.


## Usage
To install the project and use it:
- Clone the project.
- Go to the root of project. 
```
cd movies-app
```
- In the root of project, install dependencies
```
npm install
```
- To Run Application
```
npm start
```
- To run tests
```
npm test
```
